# YouTube Clone
Browse and manage your favourite videos. Reactjs.

## Installation

* [Node.js](https://nodejs.org/en/) 
* [npm](https://www.npmjs.com/get-npm) 
* [Yarn](https://www.yarnpkg.com)

```bash
  npm i -g yarn
```

## Usage

```bash
  cp .env.example .env
  yarn install
  yarn start
```
